## 项目技术栈 🛠️

- 前端：HTML5 + JavaScript + CSS3
- 游戏引擎：Pixi.js + WebGL
- 动画：GSAP + Tween.js
- 音效：Web Audio API

## 项目内容 🧧

Pixi红包雨Demo 项目，基于Pixi.js引擎开发的红包雨互动游戏。

- 红包雨效果：粒子系统、重力模拟、碰撞检测
- 交互功能：点击收集、分数统计、音效反馈
- 视觉效果：动画过渡、特效渲染、UI界面
- 性能优化：对象池、帧率控制、内存管理

## 项目亮点/成果 ✨

- 流畅的红包雨动画效果
- 高性能的粒子系统实现
- 响应式交互体验
