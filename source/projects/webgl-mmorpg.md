## 项目技术栈 🛠️

- 前端：HTML5 + JavaScript + WebGL
- 3D引擎：Three.js + WebGL
- 网络：WebSocket + Socket.io
- 后端：Node.js + Express + MongoDB

## 项目内容 🎮

WebGL MMORPG 项目，基于WebGL技术开发的多人在线角色扮演游戏。

- 3D渲染：场景渲染、角色建模、光影效果
- 多人联机：实时同步、状态管理、网络优化
- 游戏系统：角色系统、技能系统、装备系统
- 交互功能：聊天系统、组队功能、PVP对战

## 项目亮点/成果 ✨

- 流畅的3D游戏体验
- 稳定的多人联机功能
- 完整的MMORPG游戏系统
