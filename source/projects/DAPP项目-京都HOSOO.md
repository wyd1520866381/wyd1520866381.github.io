## 项目技术栈 🛠️

- 前端：React + TypeScript + Web3.js
- 区块链：Ethereum + Solidity
- NFT标准：ERC-721 + IPFS
- 图形处理：SVG + Canvas API

## 项目内容 🎨

为HOSOO打造的NFT升级DAPP，通过到店打卡或参与活动获取初始NFT，后续扫码升级NFT。

- NFT铸造：初始NFT生成、链上存储、元数据管理
- 升级机制：扫码升级、属性变更、版本控制
- 分层展示：SVG分层叠加、动态渲染、视觉效果
- 链上操作：智能合约交互、交易确认、状态同步

## 项目亮点/成果 ✨

- NFT链上操作
- 根据链上数据展示不同NFT形式（SVG分层叠加）
