## 项目技术栈 🛠️

- 前端：React + TypeScript + Ant Design Pro
- 移动端：uni-app + uView UI
- 视频流：WebRTC + FFmpeg
- 数据可视化：ECharts + D3.js

## 项目内容 ⛏️

为矿场、料场周围布控，实时把控现场情况。以摄像头、云台结合构建视频数据可视化平台。

- 视频监控：多路视频流、实时预览、录像回放
- 车辆识别：入出口抓拍、车牌识别、通行记录
- 数据统计：实时数据、报表生成、趋势分析
- 异常处理：智能报警、异常审核、处理流程

## 项目亮点/成果 ✨

- 50+摄像头稳定运行
- 入出口车辆识别抓拍
- 周报、日报、月报自动生成
- 异常报警与审核
