## 项目技术栈 🛠️

- 前端：原生小程序开发
- AR 技术：微信 AR 能力 + 自研 AR SDK
- 地图：腾讯地图 API
- 音频：微信音频 API

## 项目内容 🗺️

为旅顺文旅集团打造的全景点导游小程序，支持 AR 水瓶叠加、AR 明信片叠加、景区扫码听语音讲解，地图导航至目的地。

- AR 功能：水瓶叠加、明信片叠加、实景导航
- 语音讲解：景区扫码、自动播放、多语言支持
- 地图导航：景点定位、路线规划、实时导航
- 用户系统：个人中心、收藏功能、历史记录

## 项目亮点/成果 ✨

- AR 水瓶、明信片叠加
- 景区扫码听语音讲解
